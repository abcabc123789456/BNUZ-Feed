# BNUZ Feed · 北师珠通知聚合

北京师范大学珠海校区通知聚合平台，实时抓取 8 个官方站点的最新通知。

🔗 **在线访问**: https://your-username.github.io/bnuz-feed/

## 特性

- 📡 8 个官方站点实时抓取（教务、团委、书院、后勤等）
- 🎨 Material You 设计风格
- ⚡ 纯前端实现，无需后端服务器
- 📱 响应式布局，支持移动端
- 🔄 自动刷新，始终保持最新

## 本地开发

```bash
npm install
npm run dev
```

## 部署

项目已配置 GitHub Actions，push 到 main 分支会自动部署到 GitHub Pages。

## 技术栈

- React 18 + Vite
- Material You 设计系统
- GitHub Actions 自动部署

- 校区通知公示
- 教务部

- 国际处
